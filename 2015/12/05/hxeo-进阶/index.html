  • 更换主题，添加评论框，安装微博挂件

如何让你的博客更加美观和人性化

更换主题

1.下载主题

    ~$ cd hexo/theme
    ~$ git clone https://github.com/hexojs/hexo-theme-    light.git light
更多主题可以在https://github.com/tommy351/hexo/wiki/Themes找到。 2.配置主题 
    ~$ vim hexo/_config.yml
    ~$ theme light

3.生成并部署github

    ~$ hexo g
    ~$ hexo d

安装评论框

因为Hexo生成的是静态文件，所以评论等动态内容需要使用第三方扩展，这里推荐使用多说。
1.注册多说账号

2.配置多说

3.生成并部署github

    ~$ hexo g
    ~$ hexo d

安装微博挂件

1.去app.weibo.com获取代码

2.配置

    vim hexo/_config.ynl
    widgets:
    - weibo

3.生成并部署github

    ~$ hexo g
    ~$ hexo d
