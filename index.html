Ubuntu下MySQL和phpadmin安装教程

Ubuntu下安装MySQL

Ubuntu14.04下安装MySQL分为两种方式，一种是安装系统自带的MySQL,版本号一般为5.5.6，另一种是自定义安装，即各种版本的MySQL,官网上最新的版本为5.7.10。

第一种安装方式

由于Ubuntu14.04自带5.5.6版本的MySQL,所以安装起来十分简单，只需要几条命令即可
1.MySQL安装

    ~$ sudo apt-get install mysql-server mysql-client

安装中需要输入root（用户）密码

确认root的密码：

2.安装完毕后测试是否成功

    ~$ sudo mysql -u root -p

输入登录密码，查看是否成功。

第二种安装方式

1.在官网上下载64位的安装包，最新版本为5.7.10，网址：http://dev.mysql.com/downloads/mysql/
2.在Ubuntu Linux平台下选择mysql-server_5.7.10-1ubuntu14.04_amd64.deb-bundle.tar进行下载
3.使用命令进行解压或者右键提取

    ~$ tar -zxvf mysql-server_5.7.10-1ubuntu14.04_amd64.deb-bundle.tar

解压出如下文件
1.libmysqlclient20_5.7.10-1ubuntu14.04_amd64.deb;
2.libmysqlclient-dev_5.7.10-1ubuntu14.04_amd64.deb;
3.libmysqld-dev_5.7.10-1ubuntu14.04_amd64.deb;
4.mysql-community-client_5.7.10-1ubuntu14.04_amd64.deb;
5.mysql-community-server_5.7.10-1ubuntu14.04_amd64.deb;
6.mysql-community-source_5.7.10-1ubuntu14.04_amd64.deb;
7.mysql-client_5.7.10-1ubuntu14.04_amd64.deb;
8.mysql-community-test_5.7.10-1ubuntu14.04_amd64.deb;
9.mysql-common_5.7.10-1ubuntu14.04_amd64.deb;
10.mysql-server_5.7.10-1ubuntu14.04_amd64.deb;
11.mysql-community_5.7.10-1ubuntu14.04_amd64.changes;
12.mysql-testsuite_5.7.10-1ubuntu14.04_amd64.deb;

4.安装MySQL
首先要安装1个必须的包分别是：libaio1，使用如下命令进行安装

    ~$ sudo apt-get install libaio1

在Ubuntu终端进入到解压的文件夹，进行MySQL安装，命令如下

    ~$ sudo dpkg -i mysql-common_5.7.10-1ubuntu14.04_amd64.deb
    ~$ sudo dpkg-preconfigure mysql-community-server_5.7.10-1ubuntu14.04_amd64.deb(此步骤需要创建root密码)
    ~$ sudo dpkg -i libmysqlclient20_5.7.10-1ubuntu14.04_amd64.deb
    ~$ sudo dpkg -i libmysqlclient-dev_5.7.10-1ubuntu14.04_amd64.deb
    ~$ sudo dpkg -i libmysqld-dev_5.7.10-1ubuntu14.04_amd64.deb
    ~$ sudo dpkg -i mysql-community-client_5.7.10-1ubuntu14.04_amd64.deb
    ~$ sudo dpkg -i mysql-client_5.7.10-1ubuntu14.04_amd64.deb
    ~$ sudo dpkg -i mysql-common_5.7.10-1ubuntu14.04_amd64.deb
    ~$ sudo apt-get -f install(安装第二个必须的包libmecab2)
    ~$ sudo dpkg -i mysql-community-server_5.7.10-1ubuntu14.04_amd64.deb
    ~$ sudo dpkg -i mysql-server_5.7.10-1ubuntu14.04_amd64.deb

此时MySQL已安装完毕，可使用第一种方法中的登录命令进行登陆。

MySQL的卸载

MySQL的卸载也很简单，几行命令即可

    ~$ sudo apt-get autoremove --purge mysql-server-5.0
    ~$ sudo apt-get remove mysql-server
    ~$ sudo apt-get autoremove mysql-server
    ~$ sudo apt-get remove mysql-common
清理残留数据 
    ~$ dpkg -l |grep ^rc|awk '{print $2}' |sudo xargs dpkg -P

Ubuntu14.04中安装phpadmin

phpadmin是MySQL的图形管理界面，对于学习和使用MySQL有很大的用处。
1.安装Apache,由于Ubuntu里自带了Apache,安装起来十分方便

    ~$ sudo apt-get install apache2

安装完毕后，在浏览器中输入http://localhost/，界面显示 It works就成功了

对Apache重启的命令

    ~$ sudo /etc/init.d/apache2 restart

2.安装PHP

    ~$ sudo apt-get install php5
    ~$ sudo apt-get install libapache2-mod-php5
    ~$ sudo /etc/init.d/apache2 restart

由于版本问题现在的网页在html文件夹下，直接放在www文件夹下无法成功。当然这也可以通过配置文件来解决，具体步骤如下

    ~$ sudo gedit /var/www/html/test.php

添加如下内容：
<?php
    phpinfo();
     ?>

保存后，打开 http://localhost/test.php, 显示PHP配置信息就OK！如图

3.安装phpadmin

    ~$ sudo apt-get install phpmyadmin

安装完成，浏览器访问http://localhost/phpmyadmin并不能打开数据库管理界面，是因为phpmyadmin文件夹不在/var/www/html/文件夹下，这时候用软连接就好

    ~$ cd /var/www//html
    ~$ sudo ln -s /usr/share/phpmyadmin

再次访问http://localhost/phpmyadmin即可，这里的账号密码是指你的mysql的账号密码(用户名一般是root)。

访问phpadmin所遇到的问题

安装完phpadmin后，当你输入账号密码进入首页后，下方会提示三个问题：
1.使用配置文件中定义的控制用户连接失败。
2.phpMyAdmin 高级功能尚未完全设置,部分功能未激活。
3.缺少 mcrypt 扩展。
对于第一个第二个问题的解决方法是，对/etc/phpmyadmin/config.inc.php重新配置，修改方式如下：

    ~$ cfg['Servers'][$i]['controluser'] = '用户名';
    ~$ cfg ['Servers'][$i]['controlpassword'] = '密码';

然后重新打开配置文件将以下用//注释掉的模块都消去注释,并将文件中的模块名后边的pma_单下划线改成pma__两条下划线，保存即可，重启apache2。
第三个问题的解决方法：

    ~$ apt-get install php5-mcrypt

Gitbub pages+hexo+Markdown 搭建静态博客

关于Markdown

  • Markdown 是一种轻量级的「标记语言」，被越来越多的写作爱好者，撰稿者广泛使用。Markdown的语法十分简单，常用的标记符号不超过十个。
  • 工具推荐：小书匠。
  • 关于Markdown的语法，小书匠安装好以后，会有介绍文档。

基于Github Pages和Hexo搭建静态博客

利用Github Pages搭建静态博客的步骤

1.在github创建一个仓库，命名为：username.github.io，其中username为github账号的用户名；
2.打开git shell ，将刚才创建的仓库clone到本地

   ~ $ git clone https://github.com/username/username.github.io

3.进入仓库目录，创建index.html

   ~ $ cd username.github.io
   ~ $ echo "Hello World" > index.html

4.将修改push到远端仓库

   ~ $ git add --all
   ~ $ git commit -m "Initial commit"
   ~ $ git push -u origin master

现在，在浏览器访问http://username.github.io， 就可以看到Hello World页面了。

Hexo的安装与使用

1.Hexo是一款基于Node.js的静态博客框架，要使用Hexo，首先需要安装Node.js。Node.js的安装很简单，见Google或者百度或者Node.js官网。安装好Node.js后，就可以安装Hexo了

   ~$ npm install hexo-cli -g

2.现在，创建一个博客

   ~$ hexo init blog
   ~$ cd blog
   ~$ npm install

3.添加一篇文章

   ~$ hexo new "Hello Hexo"

4.生成静态网页文件

   ~$ hexo generate 
   or
   ~$ hexo g

5.启动本地server

   ~$ hexo server
   or
   ~$ hexo s

现在用浏览器访问http://localhost:4000， 就可以看到我们刚才创建好的博客，博客中有两篇文章，分别为Hello World（Hexo默认创建）和Helo Hexo（我们刚创建的，因为没有添加任何内容，所以为空，可通过小书匠来编辑我们的文章，然后生成静态文件，最后部署）。

将Hexo站点部署到Github Pages

1.打开git shell并cd到blog目录
2.安装hexo-deployer-git

   ~$ npm install hexo-deployer-git --save

3.配置_config.yml文件，在文件末尾添加

    deploy:
        type: git
        repo: https://github.com/username/username.github.io.git
        branch: master

4.部署站点到Github Pages

   ~$ hexo deploy

现在用浏览器访问http://username.github.io 就可以看到刚才部署的站点了。
